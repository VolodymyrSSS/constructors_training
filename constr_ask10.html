Конструкторы, задача

Какое значение будет хранить переменна obj?

var obj = Object.create(null);

Вариант1:
Объект у которого прототипом будет Object.prototype
Ответ: no: так как мы создали объект, в качестве прототипа которого не будет выступать ни один из объектов, даже глобальный прототип для всех объектов.
проверка: obj.__proto__ === Object.prototype; // false - глобального прототипа нету; также obj === null; // false
Вариант2:
Объект с полем value и значением null
Ответ: no: так как console.log(obj.hasOwnProperty); // undefined

Вариант3:
Объект у которого не будет прототипа
Ответ: no: так как obj.__proto__ === null; // false мы создали объект, в качестве прототипа которого не будет выступать ни один из объектов, даже глобальный прототип для всех объектов.

Вариант4:
null
Ответ: yes: так как obj instanceof Object; // false показало, что у объекта совсем нет прототипа, и во внутреннем поле prototype лежит значение null

Вариант5:
undefined
Ответ: no: так как console.log(obj.hasOwnProperty); // undefined - мы создали объект, в качестве прототипа которого не будет выступать ни один из объектов, даже глобальный прототип для всех объектов. И мы не получим доступ к методам из этого глобального прототипа
