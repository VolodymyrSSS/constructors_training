Конструкторы, задача

Выберите верные утверждения.

function Dog(name) {
this.name = name;
}

function Cat(name) {
this.name = name;
}

var misty = new Cat('Misty');

Вариант1:
misty instanceof Object
Ответ: yes: Несмотря на то, что функция Object не конструктор для misty, прототип по умолчанию для всех объектов Object.prototype присутствует в цепочке прототипов для misty. Прототипом для misty выступает Cat.prototype, а прототипом для него Object.prototype.

Вариант2:
misty instanceof Dog
Ответ: no: тут прототипом для misty не выступает Dog.prototype.

Вариант3:
misty instanceof Cat
Ответ: yes: тут прототипом для misty выступает Cat.prototype.
