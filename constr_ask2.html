Конструкторы, задача

Выберите верные способы привязки прототипа к создаваемым конструктором Cat объектам.

function Cat(name) {
this.name = name;
}

Вариант1:
Cat.prototype = {
sleep: function() {}
}
Ответ: no: хотя привязка сработает, но мы перезаписываем поле 'prototype' и уничтожаем в нём поле 'constructor', которое хранит ссылку на функцию Cat.

Вариант2:
Object.setPrototypeOf(Cat, {
sleep: function () {}
});
Ответ: no: так мы добавим прототип для функции, а не для создаваемых объектов

Вариант3:
Cat.__proto__.sleep = function () {};
Ответ: no: так мы добавим прототип для функции, а не для создаваемых объектов.

Вариант4:
Cat.prototype.sleep = function () {};
Ответ: yes: так как мы только дополнили методом 'sleep' при этом сохранили поле 'constructor'
