Конструкторы, задача

Какое имя созданного кота будет выведено на экран?

function Cat (name) {
this.name = null;
return {
name: name }
}

var barsik = new Cat('Barsik');

console.log(barsik.name);

Вариант1:
null
Ответ: no: мы пытаемся вернуть из конструктора примитивное значение null (или число, строку), в этом случае интерпретатор просто проигнорирует эту строку и будет работать как раньше: он будет возвращать вновь создаваемый объект

Вариант2:
Barsic
Ответ: yes: в этом случае интерпретатор проигнорирует null и возвратит вновь создаваемый объект c полем name где записано 'Barsik'

Вариант3:
Произойдёт ошибка так как this ссылается на undefined и мы не сможем обратится к полю name у undefined
Ответ: no: хотя this ссылается на поле в котором записано null, но интерпретатор проигнорирует его и возвратит создаваемый объект Cat c полем name в котором записан 'Barsik'
