Конструкторы, задача

Выберите переменные, значением которых в итоге будет объект?

function Cat(name) {
this.name = name;
}

function Dog(name) {
return { name: name };
}

function Fish(name) {
this.name = name;
return null;
}

var guppie = new Fish('Guppie');
var chappie = new Dog('Chappie');
var swittie = new Cat('Swittie');
var misty = Cat('Misty');

Вариант1: chappie
Ответ: yes: это конструктором для создания новых объектов по типу 'Dog' который на вход принимает имя этого собаки и записывает его в поле 'name'. Интерпретатор возвращает новый объект, хранящийся по ссылке this.

Вариант2: misty
Ответ: no: переданное значение имени кота не будет записано в новый объект кота, а будет записано в глобальный объект в поле name, так как, вызывая функцию, this по умолчанию будет ссылаться на глобальный объект.

Вариант3: guppie
Ответ: yes несмотря на то, что функция возвращает null, интерпретатор проигнорирует эту строку, если мы используем её как конструктор.

Вариант4: swittie
Ответ: yes: - это конструктор для создания новых объектов по типу 'Cat', который на вход принимает имя этого кота и записывает его в поле 'name'. Здесь this ссылается на имя конструктора по типу 'Cat'
