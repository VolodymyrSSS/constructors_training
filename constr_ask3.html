Конструкторы, задача

Выберите верные способы построения цепочки прототипов для объектов создаваемых конструктором Cat.

function Animal() {}
Animal.prototype.sleep = function();

function Cat(name) {}

Вариант1:
Cat.prototype = Animal.prototype;
Cat.prototype.meow = function () {};
Ответ: no: тут цепочка прототипов имеет двустороннюю направленость и метод meow появится у всех животных, а не только у кошек.

Вариант2:
Cat.prototype = Object.create(Animal.prototype);
Cat.prototype.meow = function () {};
Ответ: yes: тyt Object.create(Animal.prototype) создаёт объект, прототипом которого становится объект, переданный Animal.prototype.

Вариант3:
Cat.prototype = Object.create(Animal);
Cat.prototype.meow = function () {};
Ответ: no: В create можно передать либо объект, либо null, но не функцию.
